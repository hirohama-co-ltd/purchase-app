【新規作成】

clasp create --type sheets --title "購買申請"

【既存スプレッドシートへ紐づけ】
Apps Script の Script ID を .clasp.json に設定

2. 初回アップロード
clasp push

3. スプレッドシートで メニュー「購買申請」→「全シート＋ヘッダーを一括作成」

4. ワークフロー設定
ワークフロー設定の「アプリ別利用設定」に以下を登録
APP_CODE: PURCHASE_REQUEST

5. Webアプリとしてデプロイ
clasp deploy --description "購買申請 初回デプロイ"

6. 申請ポータル連携
ワークフロー設定の「ポータル連携」で以下を登録
アプリコード: PURCHASE_REQUEST
アプリ名: 購買申請
データ種別: purchase
ssId: 購買申請スプレッドシートID
Web URL: デプロイURL

【外貨対応 初回セットアップ】
1. スプレッドシート → メニュー「購買申請」→「全シート＋ヘッダーを一括作成」
   ※ 通貨 / 為替レート / 予想円換算 列が追加されます
2. 為替レートは GOOGLEFINANCE で取得（外部API権限不要）
3. 外貨選択時に為替レート・予想円換算が表示されることを確認

【定期購買（毎月自動申請）初回セットアップ】
1. スプレッドシート → メニュー「購買申請」→「全シート＋ヘッダーを一括作成」
   ※ 定期購買設定 / 定期購買明細 シートが追加されます
2. Apps Script エディタ → installRecurringPurchaseTrigger 実行
   ※ 毎日 7:00 JST に runRecurringPurchaseJobs が動きます
3. clasp push + 再デプロイ後、Mail スコープの再承認
4. Web UI「定期申請」タブで登録・停止を確認
